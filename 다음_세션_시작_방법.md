## ⚡ 빠른 명령어

새 Claude Code 세션을 시작한 후, 다음 중 하나를 입력하세요:

### 방법 1: Memory 검색 (권장)
```
TestCase 프로젝트 정보 알려줘
```

### 방법 2: Memory 직접 조회
```
# TestCase
```
(# 기호는 Memory 검색 키워드)

### 방법 3: 엔티티 이름으로 조회
```
"TestCase Management System" 프로젝트 정보를 Memory에서 찾아줘
```

### 방법 4: 문서 읽기
```
TestCase/docs/프로젝트_전체_요약.md 읽어줘
```

## 🎯 각 방법의 차이

### Memory 사용 시:
```
입력: "TestCase 프로젝트 정보 알려줘"

Claude가 자동으로:
1. Memory에서 "TestCase Management System" 검색
2. 관련 엔티티 5개 조회
3. 프로젝트 요약 정보 제공
```

**장점:**
- 빠르고 간단
- 핵심 정보만 제공
- 자동으로 관련 정보 연결

### 문서 읽기 시:
```
입력: "docs/프로젝트_전체_요약.md 읽어줘"

Claude가:
1. 파일 직접 읽기
2. 상세 내용 확인
```

**장점:**
- 전체 상세 정보 확인 가능
- 문서 형식 그대로 보기

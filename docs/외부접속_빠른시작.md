# ⚡ 외부 접속 빠른 시작 가이드

## 🎯 서버 시작 방법

### **가장 쉬운 방법:**
```
TestCase 폴더에서:
start-server.bat 더블클릭
```

또는

```bash
cd TestCase
npm run serve:external
```

---

## 🔧 필수 설정 (한 번만!)

### 1️⃣ AWS Security Group (필수!)

```
AWS Console 접속
→ EC2 → 인스턴스 선택
→ 하단 "보안" 탭 → 보안 그룹 클릭
→ "인바운드 규칙 편집"
→ "규칙 추가"

설정:
- 유형: 사용자 지정 TCP
- 포트 범위: 8090
- 소스: 0.0.0.0/0 (모든 IP)
- 설명: TestCase Server

→ "규칙 저장"
```

### 2️⃣ Windows 방화벽

```
setup-firewall-8090.ps1 우클릭
→ "관리자 권한으로 실행"
```

또는 PowerShell 명령어:
```powershell
New-NetFirewallRule -DisplayName "TestCase - Port 8090" -Direction Inbound -Protocol TCP -LocalPort 8090 -Action Allow
```

---

## 🌐 접속 주소

### EC2 퍼블릭 IP 확인:
```
AWS Console → EC2 → 인스턴스 → "퍼블릭 IPv4 주소" 복사
```

### 접속:
```
http://[퍼블릭_IP]:8090
```

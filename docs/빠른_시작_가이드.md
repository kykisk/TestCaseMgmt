# ⚡ 빠른 시작 가이드

## 🎯 외부 서비스를 위한 3단계

### 1️⃣ 방화벽 설정 (한 번만)
```bash
# setup-firewall.bat 파일을 마우스 우클릭
# → "관리자 권한으로 실행" 선택
```

### 2️⃣ 서버 시작
```bash
# start-production-external.bat 더블클릭
```

### 3️⃣ 접속 확인
```
브라우저에서 접속:
- 본인: http://localhost:3000
- 내부망: http://172.31.47.228:3000
- 외부: http://공인IP:3000 (포트 포워딩 필요)
```

## 📱 모바일에서 접속하기

같은 Wi-Fi에 연결된 스마트폰에서:

## 📊 현재 상태

### ✅ 준비 완료
- 프로덕션 빌드 생성 완료
- serve 웹서버 설치 완료
- 시작 스크립트 생성 완료
- IP 주소: 172.31.47.228
- 포트: 3000

### ⚠️ 필요한 작업
1. **방화벽 설정**: `setup-firewall.bat` 관리자 권한으로 실행
2. **서버 시작**: `start-production-external.bat` 실행

### 🌐 외부 인터넷 접근 (선택사항)
- 공유기 포트 포워딩 설정
- 공인 IP 또는 도메인 연결

## 💡 팁

### 자동 시작 설정
```
1. Win+R → "shell:startup" 입력
2. start-production-external.bat 바로가기 복사
3. 부팅 시 자동 실행됨
```

### 개발 서버 종료
```bash
# 현재 실행 중인 개발 서버 종료
Ctrl+C
```

### 개발 vs 프로덕션
| 항목 | 개발 (npm run dev) | 프로덕션 (serve) |
|------|-------------------|------------------|
| 포트 | 5173 | 3000 |
| 속도 | 빠른 새로고침 | 최적화된 로딩 |
| 용도 | 개발 중 | 실제 서비스 |
| 외부 접근 | 어려움 | 쉬움 |

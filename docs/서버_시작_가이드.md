# 🚀 서버 시작 가이드 (외부 접속용)

## ⚡ 빠른 시작

### **방법 1: 배치 파일 실행** (가장 쉬움!)
```
TestCase 폴더에서 start-server.bat 더블클릭
```

### **방법 2: npm 명령어**
```bash
cd TestCase
npm run serve:external
```

### **방법 3: 직접 명령어**
```bash
cd TestCase
serve -s dist -l tcp://0.0.0.0:8080
```

## 🔧 필수 설정 (한 번만)

### 1️⃣ AWS Security Group
```
AWS Console → EC2 → 보안 그룹
→ 인바운드 규칙 편집
→ TCP 8080, 소스 0.0.0.0/0 추가
```

### 2️⃣ Windows 방화벽
```powershell
# PowerShell 관리자 권한
New-NetFirewallRule -DisplayName "TestCase - Port 8080" -Direction Inbound -Protocol TCP -LocalPort 8080 -Action Allow
```

## 🛑 서버 중지

### **방법 1: 배치 파일**
```
stop-server.bat 더블클릭
```

### **방법 2: 터미널에서**
```
Ctrl + C
```

### **방법 3: 강제 종료**
```bash
# 포트 8080 프로세스 찾기
netstat -ano | findstr :8080

# PID로 종료
taskkill /F /PID [PID번호]
```

---

## 🔄 서버 재시작

```bash
# 1. 중지
stop-server.bat

# 2. 시작
start-server.bat
```

### 개발 서버
- **포트**: 5173
- **시작**: `start-dev.bat` 또는 `npm run dev`
- **DB**: dev_prefix (localStorage)
- **특징**:
  - Hot Module Replacement (자동 새로고침)

- **시작**: `start-server.bat` 또는 `npm run serve:external`
- **DB**: prod_prefix (localStorage)
- **특징**:
  - 프로덕션 빌드 (최적화됨)
  - [DEV] 배지 없음
  - 외부 접속 가능
- **환경**: AWS EC2
  - 프라이빗 IP: 172.31.47.228
  - 퍼블릭 IP: (Security Group 설정 필요)
- **용도**: 실제 서비스/데모

## 🔗 데이터 관계

```
Project (1)
  └─ Requirements (N)
  └─ TestCases (N)
       └─ Links to → Requirements (N:M)
```

**요구사항과 테스트케이스는 N:M 관계**
- 1개 테스트케이스 → 여러 요구사항 검증 가능
- 1개 요구사항 → 여러 테스트케이스로 검증 가능
